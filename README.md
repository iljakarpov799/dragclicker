# DragClicker

**DragClicker** (Drag Click Limit) — экспериментальная утилита с графическим интерфейсом для Windows. Она отслеживает вращение колёсика мыши, вычисляет на его основе частоту кликов (CPS) и эмулирует реальные клики левой или правой кнопкой мыши с этой частотой. Программа работает в фоне, глобально перехватывая события мыши через [`pynput`](https://github.com/moses-palmer/pynput).

> ⚠️ **Дисклеймер.** Утилита создана в учебно-экспериментальных целях. Использование автокликеров может нарушать правила игр и сервисов — применяйте на свой страх и риск.

---

## Возможности

- 🎯 Перевод скорости вращения колеса в частоту кликов (CPS) по линейной интерполяции.
- 🖱️ Эмуляция настоящих кликов ЛКМ или ПКМ выбранной частотой.
- 📈 Сглаживание CPS (EMA), инерция при остановке колеса и случайный джиттер для реалистичности.
- 🚫 Подавление системной прокрутки, пока программа активна (Windows, низкоуровневый хук).
- ⌨️ Глобальное управление боковыми кнопками мыши — работает поверх любых окон.
- 🌙 Тёмный интерфейс с живым графиком CPS и статистикой сессии.
- 🎚️ Настройки (макс. CPS, джиттер, инерция) меняются ползунками на лету.

## Установка

Требуется **Python 3.8+**.

```bash
git clone https://github.com/<ваш-аккаунт>/DragClicker.git
cd DragClicker
pip install -r requirements.txt
```

## Запуск

```bash
python dclimit.py
```

> Для подавления системной прокрутки на Windows программу может потребоваться
> запустить **от имени администратора** — иначе блокировка скролла и генерация
> кликов колесом не будут работать.

## Управление

| Действие | Эффект |
|---|---|
| **X1 + X2** (две боковые кнопки одновременно) | Включить / выключить программу |
| **ЛКМ** | Режим LMB (повторный клик → OFF) |
| **ПКМ** | Режим RMB (повторный клик → OFF) |
| **Зажать X2** | Временно вернуть обычный скролл (клики не генерируются) |

Режимы LMB и RMB взаимоисключающие. В окне продублированы кнопки: «ВКЛ/ВЫКЛ»,
«Режим LMB», «Режим RMB», «Режим OFF».

### Поведение прокрутки и кнопки X2

Пока программа **запущена**:

- системная прокрутка колесом **блокируется**, а вращение колеса **генерирует клики**;
- если **зажать вторую боковую кнопку (X2)** — прокрутка временно **разблокируется**
  и колесо работает как обычный скролл, клики при этом не генерируются, а программа
  **остаётся включённой**;
- когда программа выключена — прокрутка работает как обычно.

## Как считается CPS

- Скорость колеса измеряется в скользящем окне 150 мс (тик/с).
- Линейная интерполяция: 5 тик/с → 5 CPS, 100 тик/с → максимум (по умолчанию 50).
- Сглаживание EMA (α = 0.3), инерция при остановке колеса, случайный джиттер ±CPS.
- Параметры (макс. CPS, джиттер, инерция) настраиваются ползунками в реальном времени.

## Сборка `.exe`

```bash
pip install pyinstaller
pyinstaller --onefile --noconsole --name DragClicker --icon ico.ico --add-data "ico.ico;." dclimit.py
```

Готовый `DragClicker.exe` появится в папке `dist/`.

> На не-Windows платформах в `--add-data` используйте разделитель `:` вместо `;`.
